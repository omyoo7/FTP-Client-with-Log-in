# FTP-Client-with-Log-in
Connect to FTP server Log-in using user_id, Password and use to apply ftp command. 
compile using: 
g++ -g ftp_client.c -o ftp
And run using following command:
./ftp 192.168.0.1
enter user name.
enter password.
you logged in.
